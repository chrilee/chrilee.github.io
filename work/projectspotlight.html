Project Spotlight

Video Interview Platform

Role

Visual / Tech Designer

Objective

Infusion wanted a Personif instance for Infusion people to talk about cool projects that they are currently working on. The ask was to design the site and also style it using CSS. Personif is a marketing platform that allows people to communicate with the company using video.

Solution

The design of the site isnâ€™t as flexible as most projects because Personif has a set template to follow but can be modified to a certain extent with CSS. I wanted to keep the design clean so I focus a lot of it on typography with bright text and dark background.

For the styling portion of the site, there was a big learning curve as I was still new to Personif. There was a lot of backend work that was intertwined with the CSS and it took me a while to understand where everything was. Most of time was spent with font and background colours because the site was not coded with the intention of a dark background so some text was not visible in certain parts. At the end of the exercise, I was quite familiar with what files I can edit and what is customizable on Personif.
